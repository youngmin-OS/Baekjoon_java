# 📘 Baekjoon Java 문제 풀이 저장소

Java로 푼 백준(BOJ) 알고리즘 문제들을 난이도별로 정리한 저장소입니다.  
꾸준한 문제 풀이 기록과 복습, 그리고 코딩테스트 대비를 위해 운영하고 있습니다.

---

## 📁 폴더 구조

src/
├─ Bronze/ # 브론즈 문제 풀이
└─ Silver/ # 실버 문제 풀이


각 폴더에는 문제 번호와 문제명을 기준으로 Java 파일을 저장합니다.

## 📌 Commit 컨벤션

문제마다 다음 형태로 커밋 메시지를 작성합니다:

- 문제명(문제번호) -

## 📚 풀이 언급 방식

문제 Java 파일명은 다음 형태로 구성됩니다:

- `FindAlphabet.java`  → BOJ 10809  
- `YourAchieve.java`   → BOJ 25206  
- `ReverseBasket.java` → BOJ 10811
    
파일명은 문제명을 기반으로 작성하며, 폴더는 난이도로 분류합니다.
